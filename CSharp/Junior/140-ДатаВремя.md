# Работа с датой и временем

- Типы данных для работы с датой и временем

  - `DateTime` - Дата и время
  - `DateOnly` - Дата без времени
  - `TimeOnly` - Только время без даты
  - `TimeSpan` - Интервал времени
  - `DateTimeOffset` - Дата и время относительно UTC (с часовыми поясами). Предпочтительный тип данных для систем, работающих для клиентов во многих часовых поясах. (Боты, API и др.)
  - `TimeZoneInfo` - Информация о часовом поясе

  Так же, помимо обывательского времени .Net имеет понятие "тики" (англ. Ticks). В 100 наносекундах 1 тик. DateTime хранит количество таких тиков с полуночи 1 января 0001 года. Для конвертации тиков в другие единицы используется тип `TimeSpan` и его поля: `TicksPerDay`, `TicksPerHour`, `TicksPerMinute`, `TicksPerSecond`.

- Создание объекта для работы со временем и получение текущего времени.

    ```csharp
      using System;
      using System.Globalization;

      DateTime now = DateTime.Now;
      Console.WriteLine($"Текущее время: {now}");

      DateTime utcNow = DateTime.UtcNow;
      Console.WriteLine($"UTC: {utcNow}");

      DateTimeOffset nowWithOffset = DateTimeOffset.Now;
      Console.WriteLine($"Местное время: {nowWithOffset} часовой пояс: {nowWithOffset.Offset}");

    ```

    Так же можно создавать и использовать экземпляры класса `DateTime` с определённой датой и временем:

    ```csharp

      // Только дата yyyy, mm, dd (Время 00:00:00)
      DateTime birthday = new DateTime(1999, 10, 30);

      // Дата и время (часы, минуты, секунды)
      DateTime meeting = new DateTime(2026, 6, 15, 14, 30 , 0);

    ```

    Дата и время позволяют вычислять интервалы времени:

    ```csharp

     DateTime start = new DateTime(2022, 10, 1, 11, 0, 0);
       DateTime end = new DateTime(2026, 6, 11, 12, 44, 0);
           TimeSpan diffTime = end - start;
       
       Console.WriteLine($"Интервал: {diffTime}"); // 1349.01:44:00
       Console.WriteLine($"Дней: {diffTime.Days}");          // Дней: 1349
       Console.WriteLine($"Всего дней: {diffTime.TotalDays}"); // Всего дней: 1349.07222222222
       Console.WriteLine($"Всего часов: {diffTime.TotalHours}"); // Всего часов: 32377.7333333333
       Console.WriteLine($"Всего минут: {diffTime.TotalMinutes}"); // Всего минут: 1942664
    
    ```

    Операции сравнения с датами:

    ```csharp
    
      DateTime date1 = new DateTime(2026, 6, 1);
      DateTime date2 = new DateTime(2026, 6, 15);

      // Compare
      int result = DateTime.Compare(date1, date2);
      // result < 0 -> date1 раньше date2
      // result = 0 -> даты равны
      // result > 0 -> date1 позже date2

      //CompareTo
      int compareResult = date1.CompareTo(date2); // -1, 0, 1
      
      // Equals
      bool isEqual = date1.Equals(date2); 

      // Сортировка с использованием сравнения:
      List<DateTime> dates = new List<DateTime> {date2, date1};
      dates.Sort((a,b) => DateTime.Compare(a, b));
      // или dates.Sort(); т.к. DateTime реализует IComparable
    
    ```

- Форматирование даты и времени. Рассмотрим только основные типы даты и времени: американский формат, европейский, российский.
  
    ```csharp
      using System.Globalization;
      // ...
      DateTime now = DateTime.Now;

      Console.WriteLine(now.ToString("d")); // Короткий формат даты 6/11/2026
      Console.WriteLine(now.ToString("D")); // Полный формат даты Thursday, June 11, 2026
      Console.WriteLine(now.ToString("t")); // Америконское время 10:20 AM
      Console.WriteLine(now.ToString("T")); // Америконское время с секундами 10:20:06 AM
      Console.WriteLine(now.ToString("f")); // Америконская дата Thursday, June 11, 2026 10:20 AM
      Console.WriteLine(now.ToString("F")); // Америконская дата с секундами Thursday, June 11, 2026 10:20:06 AM
      Console.WriteLine(now.ToString("g")); // Короткая америконская дата и америконское время 6/11/2026 10:20 AM
      Console.WriteLine(now.ToString("G")); // Оно же с секундами 6/11/2026 10:20:06 AM
      Console.WriteLine(now.ToString("s")); // Ещё и всрато-инвертированная дата 2026-06-11T10:22:19
      Console.WriteLine(now.ToString("u")); // Ещё одна всратая вариация 2026-06-11 10:22:19Z
      Console.WriteLine(now.ToString("r")); // Thu, 11 Jun 2026 10:22:19 GMT
      // Ручное форматирование даты и времени:
      Console.WriteLine(now.ToString("HH:mm:ss dd.MM.yy")); // 10:24:13 11.06.26

      // Культурно-зависимое форматирование:
      var ukCulture = new CultureInfo("en-GB");
      var usCulture = new CultureInfo("en-US");
      var ruCulture = new CultureInfo("ru-RU");
      Console.WriteLine(now.ToString(ukCulture)); // 11/06/2026 10:26:56
      Console.WriteLine(now.ToString(usCulture)); // 6/11/2026 10:26:56 AM
      Console.WriteLine(now.ToString(ruCulture)); // 11.06.2026 10:26:56
    
      // Так же можно интерполировать данные даты и времени
      Console.WriteLine($"{now:MMMM dd, yyyy}"); // June 11, 2026
    ```

- Парсинг даты и времени

    `DateTime.Parse` и `DateTime.TryParse` позволяют преобразовать распространённые форматы даты и времени в объекты `DateTime`

    ```csharp
    string dateInput = "14:40 15.06.2026";
    CultureInfo culture = new CultureInfo("ru-RU");
    // Parse - ошибка, если формат некорректный
    DateTime parsedDate = DateTime.Parse(dateInput, culture);
    Console.WriteLine(parsedDate.ToString(culture)); // 15.06.2026 14:40:00

    // Рекомендуемая форма парсинга данных - TryParse, поскольку можно проверить, валидные ли данные пришли.
    DateTimeStyles style = DateTimeStyles.None;
    if (DateTime.TryParse(dateInput, culture, style, out DateTime result)){
      Console.WriteLine($"Successfuly: {result}");
    }else{
      Console.WriteLine("Incorrect date format");
    }
    
    // Если необходимо контроллировать формат даты, который примет программа DateTime.TryParseExact(...)
    string[] formats = {"dd.MM.yyyy", "HH:mm dd.MM.yyyy"};
    if (DateTime.TryParseExact(dateInput, formats, CultureInfo.InvariantCulture, DateTimeStyles.None, out DateTime result)){
      Console.WriteLine($"Successfuly: {result}");
    }else{
      Console.WriteLine("Incorrect date format");
    }
    ```

- Работа с часовыми поясами:
  
    ```csharp

        DateTimeOffset utcTime = DateTimeOffset.Now;
        DateTimeOffset birthday = new DateTimeOffset(1999, 10, 30, 12, 15, 0, TimeSpan.Zero);

        Console.WriteLine($"UTC Time with offset: {utcTime}");
        Console.WriteLine($"UTC: {utcTime.UtcDateTime}");
        Console.WriteLine($"UTC Time offset : {utcTime.Offset}");

        DateTimeOffset localTime = utcTime.ToLocalTime(); // Системные настройки часового пояса позволяют данному методу определить локальное время
        Console.WriteLine($"LocalTime: {localTime}");

        // Для API и баз данных рекомендуется использовать UTC время без смещения:
        DateTime toDB = utcTime.UtcDateTime;
        DateTimeOffset fromDB = new DateTimeOffset(toDB); // Преобразование в обратную сторону, часовой пояс +0

        // TimeZoneInfo - работа с часовыми поясами. 
        TimeZoneInfo localZone = TimeZoneInfo.Local;
        TimeZoneInfo utcZone = TimeZoneInfo.Utc;
        TimeZoneInfo helsinkiZone = TimeZoneInfo.FindSystemTimeZoneById("Europe/Helsinki");

        DateTime converted = TimeZoneInfo.ConvertTimeFromUtc(toDB, helsinkiZone);
        Console.WriteLine($"Helsinki Time Now: {converted}");
        converted = TimeZoneInfo.ConvertTimeFromUtc(toDB, utcZone);
        Console.WriteLine($"UTC Time Now: {converted}");
        converted = TimeZoneInfo.ConvertTimeFromUtc(toDB, localZone);
        Console.WriteLine($"Local Time Now: {converted}");

    ```
